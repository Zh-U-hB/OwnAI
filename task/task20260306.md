# 开发任务计划

## agent实例创建

### src/agent/agent_register.py

这个组件定义了Agent的注册方法，负责创建、删除、列出所有Agent实例。
三个函数：

  - [ ] 创建Agent实例的方法
  - [ ] 删除Agent实例的方法
  - [ ] 列出所有Agent实例的方法

### src/agent/abilities/base.py

这个组件定义了所有Agent的基本能力，包括文件读写、终端命令执行、子代理创建、浏览器搜索、agent间通讯等。

- [ ] 实现BaseAbility类
  - [ ] 初始化方法
  - [ ] 工具方法

### src/agent/abilities/permission.py

这个组件作为权限设置工具，负责设置Agent权限等级，这个文件中包括：
- [ ] 定义工具的权限等级类型
- [ ] 定义文件夹权限等级类型

### src/agent/abilities/serve.py

这个组件定义了Agent的abilities接口，规定了Agent根据Agent设置能够执行的能力方法。
这个组件需要调用BaseAbility类的工具方法以及PermissionAbility类的权限方法。

- [ ] 实现ServeAbility类
  - [ ] 初始化方法
  - [ ] 执行能力方法

### src/agent/abilities/exec.py

这个组件定义了Agent的执行能力，包括执行终端命令、执行Python代码等。

- [ ] 实现ExecAbility类
  - [ ] 初始化方法
  - [ ] 执行能力方法

### src/agent/abilities/process.py

这个组件定义了LLM回复的处理能力，包括解析LLM回复、提取信息、格式化回复等。

- [ ] 实现ProcessAbility类

  - [ ] 执行能力方法
